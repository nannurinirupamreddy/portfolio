PlutoPay - A Secure Java-Based Payment System

PlutoPay is a secure, console-based P2P payment system developed in Java. It allows users to register, log in, manage balances, send and request money, and view transaction history. Additional features include PIN-protected transactions, virtual card issuance and sharing with CVV verification, a favorite account list, and a reward tracking system. Data is stored persistently using JSON and managed through Java File I/O, with built-in security checks for sensitive actions and failed attempts. The project highlights strong object-oriented design and secure file-based state management.

BeatScape: Playlist Simulator

Playlist Simulation System is a Java-based music player simulator that utilizes the Linked List data structure to manage playlists dynamically. It allows users to create, update, and modify playlists while storing all data in a MySQL database. The system leverages JDBC for database connectivity and follows core Object-Oriented Programming (OOP) principles to ensure modularity and scalability. This project demonstrates effective use of data structures, database integration, and OOP design in a real-world simulation.

Airline Booking Waitlist Simulation

Airline Booking System is a Java-based application that simulates real-world airline booking processes, inspired by systems used by airlines like Delta and Emirates. It efficiently manages seat bookings and waitlists using Linked Lists for passenger tracking and Queues to implement a FIFO-based waitlist system. The program automatically assigns seats to waitlisted passengers as slots become available and provides clear visibility into booked passengers. Built using Java (JDK 17+) and following OOP principles, the system showcases practical use of data structures in simulating real-life logistics.
